## ğŸ¨ Customization Guide

### Change Primary Color (Cyan)
Edit in `src/index.css`:
```css
:root {
  --color-primary: #00D9FF;  /* Your color here */
}
```

### Change Accent Color (Purple)
```css
:root {
  --color-accent: #7C3AED;  /* Your color here */
}
```

### Adjust Spacing
Modify margins/padding in component CSS files

### Modify Animations
Edit `@keyframes` definitions or timing values
